# ねこスロ5
## 基本仕様
- ボーナス+BTタイプのゲームです

## シンボル
- cat5（ベル）
- ファラフェル（スイカ）
- ペトリ皿（チェリー）
- 赤7
- 青7
- BAR
- REPLAY

## 配当表
- SUPER BIG
    - 赤7揃い
    - 400枚 + BT
- BIG
    - 青7揃い
    - 220枚
- REG
    - 赤7赤7BARまたは青7青7BAR
    - 96枚
- ﾍﾟﾆｭﾌﾟﾋﾟﾘｭﾘｭﾘｭﾐﾋﾟﾋﾟｭﾎﾟｨﾎﾟﾋﾟﾘｨ（チャンスモード。通常時のみ発生）
    - 赤7赤7青7または青7青7赤7
    - 90枚 + BT
- BT
    - BAR揃い 200枚
- cat5
    - 10枚
- ペトリ皿（any any ペトリ皿 で成立）
    - 2枚（中段停止時は1枚）
- ファラフェル
    - 6枚
- REPLAY
