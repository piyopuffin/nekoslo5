## モード遷移

### 通常モード（Normal）
- デフォルトのモード
- 毎スピンで一定確率でチャンスモードへ突入（設定1: 0.1%、設定6: 0.4%）
- ボーナス当選役が成立するとボーナスモードへ突入
- 1000ゲーム消化（天井）でチャンスモードへ強制突入
- ボーナス成立時に天井カウンターはリセット

### チャンスモード（ﾍﾟﾆｭﾌﾟﾋﾟﾘｭﾘｭﾘｭﾐﾋﾟﾋﾟｭﾎﾟｨﾎﾟﾋﾟﾘｨ）
- 10ゲーム以内に赤7赤7青7 or 青7青7赤7を揃えればBTへ突入（+90枚）
- チャンスモード中はchance_mode役の当選確率が50%
- 10ゲーム以内に揃わなければ通常モードへ戻る

### ボーナスモード（Bonus）
- SUPER BIG BONUS（赤7揃い）: 最大100スピン/最大獲得400枚 → 終了後BTへ
- BIG BONUS（青7揃い）: 最大60スピン/最大獲得220枚 → 終了後通常モードへ
- REG BONUS（赤7赤7BAR/青7青7BAR）: 最大30スピン/最大獲得96枚 → 終了後通常モードへ
- 累計獲得枚数が上限に達するとボーナス終了

### BTモード（ボーナストリガー）
- BAR揃いで200枚獲得 → 通常モードへ
- 赤7揃い or 青7揃いでSUPER BIG BONUS突入
- 最大100スピンで終了 → 通常モードへ
